FoodRescue - Food collection and Distribution platform

Project Overview:
FoodRescue is a web-based platform designed to reduce food waste and fight hunger by connecting restaurants with surplus food to individuals or NGOs  in need.
The platform  acts as a bridge between food donors and food receivers, ensuring proper coordination and distribution.

Purpose:
Everyday, large amount of food are wasted while many people struggle with hunger.

FoodRescue aims to:
1. Reduce food Wastage
2. Provide food to needy individuals
3. Create an organized system for food donation and distribution
4. Improve transparency in food sharing

Features:
1. User Registration and Login
    - Separate access for restaurants and food receivers
    - Basic aunthentication System
2. Food Donation Posting:
    -Restaurants can post surplus food details 
    - Including food name, quantity, and pickup time
3. Food Request Posting 
    - NGOs or individuals can request food 
    - Includes location and requirement details
4. View available food 
    - displays available food donations
    - Shows availability status
5. Admin Control
    - view all donations and requests
    - manage user and food posts
6. Status Update
    - Food dtatus can be updated as:
        - Available
        - Collected
        - Distributed
7. REsponsive Design
    - works on mobile and desktop
    - clean and simple UI
8. Secure data handling
    - stores user and food data securely
    - prevents data loss
